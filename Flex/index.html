Grid arbeitet mit verschiedenen Einheiten

px : Ist eine feste Größe, die sich nicht ändert/anpasst.

% : Ist die Prozentuale größe des Abschnittes innerhalb des Eltern-Elementes.

fr : Eine Fraktion bezieht sich auf den zur Verfügung stehenden Platz. So nimmt sich 1fr alleine immer der Gesamte Container, während sich 2fr 1fr die Platz untereinander aufteilen 2:1 (Gesamt 3 Teile) 100%/3 und dem ersten fr gehören 2 Anteile (2fr), während dem hinteren fr nur 1 Anteil gehört (1fr).

auto : Bei dieser Funktion

Die minmax Funktion Leicht erklärt

Die minmax Funktion ist die grid version von shrink und grow.

Analysieren wir mal den genauen Command:
grid-template-rows: minmax(Minbreite, Maxbreite in fr) Alle weiteren fr;
grid-template-rows: minmax(100px, 1fr) 1fr 2fr;
Wenn eine Seite auf 3 Spalten besteht, die erste sich aber auf dem Desktop verbreitern soll und auf dem Handy zusammenschieben, dann benutzt man minmax.
